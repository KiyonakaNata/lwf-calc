# Lazy Witch's Factory 生産計算機（非公式）

ゲーム **Lazy Witch's Factory** の生産計算機です。契約・レリック・バフ・ポータルの効果を
ON/OFF・重ねがけ・配置数で切り替えると、全レシピの
**生成個数 / 所要時間 / 生産速度 / 材料消費 / 買取価格** が補正後の値で一覧表示されます。

🔗 **公開ページ**: <https://kiyonakanata.github.io/lwf-calc/>（GitHub Pages）


## 対象バージョン
- **ver 0.18.1** 時点のデータ
- ストアページ: <https://store.steampowered.com/app/3971650/Lazy_Witchs_Factory/>

## 使い方
- 左の **契約スキル / レリック / バフ / ポータル** で回数（重ねがけ）や ON/OFF を設定。
- 各レシピのカード右上の **配置数** を変えると、`/秒` の値（生産速度・消費/秒・取得/秒）が台数分に増えます。
- **運搬上限**（水流コンベア 10/s・激流コンベア 40/s）を生産速度が超えると橙／赤で警告。
  「水冷クラフト」を入れるとこの上限が下がります。

## データの編集（各値を自分で弄れます）
ページ下部の **「データを編集 / 確認（JSON）」** を開くと、全アイテム・全スキルのデータが JSON で表示されます。
数値（生成個数・所要時間・材料・買取価格・スキル効果など）を書き換えて **「読み込む」** を押すと、
その場で計算結果に反映されます。「初期データに戻す」で元に戻せます。
※ 編集はブラウザ内だけの一時的なもので、ページを再読み込みすると初期データに戻ります。
