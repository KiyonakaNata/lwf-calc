# Lazy Witch's Factory 生産計算機（非公式）

ゲーム **Lazy Witch's Factory** の生産計算機です。契約・レリック・バフ・ポータルの効果を
ON/OFF・重ねがけ・配置数で切り替えると、全レシピの
**生成個数 / 所要時間 / 生産速度 / 材料消費 / 買取価格** が補正後の値で一覧表示されます。
**返済シミュレーター** で返済ノルマの到達時間を試算でき、**再現モード** では
マップ生成・土地購入・契約抽選・注文・ポータル・杖まで含む 1 ゲーム（30分）をまるごとシミュレーションできます。

## 画面構成
- **左カラム（固定）**: 契約スキルとレリック・バフの設定。タブを切り替えても常に表示されます。
  - **契約スキル（重ねがけ回数）**: 各契約の取得回数を入力。`適用順` の ▲▼ で計算順を並べ替え可。着火・24時間シフトなど**1回のみ取得できる契約は ON/OFF トグル**です。有効中のバフはリスト上部にチップ表示され、「レリック・バフ ▼」「▲ 上へ」ボタンで下のレリック・バフ枠と行き来できます。
  - **レリック・バフ**: 認定証・リトマス紙・酸性触媒の**所持数**（契約ぶん＝契約回数×付与数 ＋ 注文/手動入手の合算）と、分別不要・猛毒バフの**有効/無効**（契約または注文/手動チェックで有効化。効果は重ねても1回ぶん）。**注文/手動ぶんは返済の進行度に数えません**（契約ぶんのみカウント）。
- **右カラム（タブ切替）**:
  - **レシピ計算** … 全レシピの補正後ステータス一覧。
  - **返済シミュレーター** … 返済ノルマと必要時間の試算。
  - **⛏ 採掘・原料** … ゴブリン採掘の収入と原料収支の詳細版。
  - **🎮 再現モード** … 1ゲーム（30分）の再現シミュレーター。予約（配置・土地購入・注文・契約）⇔「▶次のイベント」で時間を進めて完済タイムを詰める。🏆リザルトと自己ベスト（難易度別）付き。**📋 RUNコード**でrun全体を1つのコードとして書き出し・読み込みでき、他人のrunの完全再現やバグ報告に使えます。ゲームとの違い（物流の省略など）はタブ内ヘルプ「⚠ ゲームとの明確な違い」を参照。

## レシピ計算タブ
- 左の **契約スキル / レリック / バフ / ポータル** で回数（重ねがけ）や ON/OFF を設定すると、全レシピが補正後の値に一斉更新されます。補正は **取得順（適用順）に上から逐次計算**されます。
- 各レシピのカード右上の **配置数** を変えると、`/秒` の値（生産速度・消費/秒・取得/秒）が台数分に増えます。
- **運搬上限**（水流コンベア 10/s・激流コンベア 40/s）を生産速度が超えると橙／赤で警告。「水冷クラフト」を入れるとこの上限が下がります。

## 返済シミュレータータブ
左の契約スキル設定（現金収入）と難易度から、返済ノルマと到達時間を計算します。
- **難易度 / 融追加資**: 返済額の式（初期額・毎回加算・5回ごと+500・返済必要回数・総額）を難易度の横に表示。融追加資で返済回数が増えます。
- **搬出上限**: 全体入力（空欄＝無制限、プリセット 10/40）＋ レシピごとの個別上限（個別が優先）。上限を超えるぶんは生産が頭打ちとして収入に反映。
- **返済に使うレシピ**: 現金を生むレシピを選択・台数指定すると、合計の現金/秒を算出します。
- **中間素材を自作 / 余剰製造**: チェックした中間素材を「作るが納品しない」素材として材料へ**再帰展開**。体数指定で供給が頭打ちになり、その素材を使う製造物だけが**律速**。端数は**余剰**として資源収支に黒字表示（「余剰は売却(納品)に回す」ONで収入へ加算）。必要な使い魔の体数も内訳付きで表示します。
- **採掘収入（契約「メンテナンス」）**: 採掘ゴブリンのクリティカル採掘で得る現金を**合計収入・到達時間に合算**します。式は `ゴブリン体数 ×(採掘速度 ÷ 必要打数3)× クリティカル率 × 砥石所持数`。
  - 採掘速度・クリ率は左パネルの採掘系契約/ポータル（メンテナンス＝砥石+1／キツツキ打法＝速度×4・クリ×0.25／太極拳＝速度×0.5／訓練所＝速度+100%/回）から自動算出（**「×N」＝乗算・「+N%」＝加算**）。
  - 枠内で **ゴブリン体数 / 砥石 手動** と、**ゴールドラッシュ**（クリ率100%）・**フォーム指導**（速度×8）・**早退**（未召喚数×10%の速度増）を指定できます。
- **資源収支**: 選択レシピ＋自作中間素材の「買取−材料」を資源ごとに合算し、黒字（余剰）/赤字（消費）を別々に表示。
- **進行度**: 取得契約数（左パネルの回数合計、ポータルと注文入手は除外。猛毒・分別不要も契約ぶんのみカウント）＋スキップ補正。これを現在の進行度として、**次の返済目標 / 次回返済額の倍率 / 次の返済まで / 合計収入** をカード表示。
- **返済表**: 各回の `返済額 / 倍率（前回比）/ 所要 / ◀から到達（現在地から貯めるのに要する累計時間）` を一覧。

## データの編集（各値を自分で弄れます）
ページ下部の **「データを編集 / 確認（JSON）」** を開くと、全アイテム・全スキルのデータが JSON で表示されます。
数値（生成個数・所要時間・材料・買取価格・スキル効果など）を書き換えて **「読み込む」** を押すと、
その場で計算結果に反映されます。「初期データに戻す」で元に戻せます。
※ 編集はブラウザ内だけの一時的なもので、ページを再読み込みすると初期データに戻ります。
